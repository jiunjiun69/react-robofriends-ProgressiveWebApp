# robofriends-pwa
adding PWA capabilities to a react app

## To run the project:

1. Clone this repo
2. Run `npm install`
3. Run `npm start`

## if change
`npm run build`

## deploy:
```
git init
git remote add origin <repo_link>
npm run deploy
```
